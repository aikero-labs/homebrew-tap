# aikero-labs Homebrew Tap

Homebrew formulae for [aikero-labs](https://github.com/aikero-labs) tools.

## Usage

```bash
brew install aikero-labs/tap/ssh-image-paste
```

Or tap first, then install:

```bash
brew tap aikero-labs/tap
brew install ssh-image-paste
```

## Available formulae

| Formula | Description |
|---------|-------------|
| [`ssh-image-paste`](https://github.com/aikero-labs/ssh-image-paste) | Paste clipboard images to remote SSH servers from macOS |
